igm.utils.pythonpath¶

with_pythonpath¶

igm.utils.pythonpath.with_pythonpath(*path, recover=True, recover_when_replaced=False)[source]¶
Overview:

Append PYTHONPATH in context, the packages in given paths will be able to be imported.

Parameters:

  • path – Appended python path.

  • recover – Recover the sys.path when context is over, default is True.

  • recover_when_replaced – If sys.path is replaced again, recover it or not, default is False.
