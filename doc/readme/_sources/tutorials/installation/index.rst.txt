Installation
===================

igm is currently hosted on PyPI. It required python >= 3.7.

You can simply install igm from PyPI with the following command:

.. code:: shell

    pip install igm

You can also install with the newest version through GitHub:

.. code:: shell

    pip install -U git+https://github.com/igm4ai/igm@main

script:
