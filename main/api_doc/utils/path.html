igm.utils.path¶

normpath¶

igm.utils.path.normpath(path: str, *paths: str) → str[source]¶
Overview:

Normalize the path to a unique format which is comparable by ==.

Parameters:

  • path – First path segment.

  • paths – Following segments.

Returns:

Full absolute path.

Examples::
>>> from igm.utils import normpath
>>> normpath('./file')
'/home/user/igm/file'
>>> normpath('/home/file')
'/home/file'
>>> normpath('~/file')
'/home/user/file'
